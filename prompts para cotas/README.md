# G3Plugins

G3Plugins é um conjunto de plugins para Autodesk Revit desenvolvido em C# utilizando a Revit API.

O objetivo do projeto é fornecer ferramentas para aumentar a produtividade em projetos estruturais, com foco em automação, organização e facilidade de uso.

## Funcionalidades

### Cotagem Automática (MVP)

- Seleção de parede
- Detecção automática das extremidades
- Detecção automática de portas
- Detecção automática de janelas
- Criação automática de cadeia de cotas

## Tecnologias

- C#
- .NET Framework 4.8
- Autodesk Revit API 2025
- Visual Studio 2022
